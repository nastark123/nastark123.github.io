Project Description

My senior design project is to create a radio microcontroller using the Efabless Skywater 130 nm Process. The microcontroller will include a RISC-V processor, generated SRAM, external peripherals such as SPI, UART, and I2C, internal peripherals such as DMA, and a PLL for generating RF frequencies. The project will not result in all of the functionality necessary to implement a radio protocol such as Zigbee or Bluetooth due to time constraints, but will be continued by future students who will build off the work of my group. My focus on the project will be creating and testing various digital peripherals, such as a crossbar for bus arbitration, the RISC-V processor, and the DMA engine. The project is still ongoing and is expected to complete in May 2025. After the microcontroller is fabricated, it will be used by the Iowa State ChipForge cocurricular, which aims to teach students about the basics of ASIC design, to see what is possible in the process and to teach new club members how to design and test ASICs.

The primary skills learned in taking on this project so far have been the Verilog language, OpenRAM RAM compiler, GTKWave for viewing testbench waveforms, and writing C files to interact with custom memory-mapped peripherals. Many of the concepts used for this have been taught in some form in other classes at Iowa State, so much of my learning is centered around applying a lot of previous concepts together to create a more complex and capable product.
